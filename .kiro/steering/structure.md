# Project Structure

```
/
├── Books/                          # Book notes and implementations
│   ├── Deep Learning for Coders with fastai and PyTorch/
│   └── Programming PyTorch for Deep Learning - Ian Pointer/
│       └── Chapter N - Topic/      # Per-chapter notebooks and notes
│
├── Classes/                        # University course materials
│   ├── Johns Hopkins/Sem N/        # Semester-organized coursework
│   ├── Stanford/                   # Stanford online courses (CS234, CS224n)
│   ├── MIT/                        # MIT courses (6.006 Algorithms)
│   └── Harvard/
│
├── Courses/                        # Online course materials
│   ├── Coursera/                   # TensorFlow, Deep Learning AI
│   ├── Scrimba/                    # React, Tailwind CSS
│   ├── Udemy/                      # React course
│   └── freeCodeCamp - D3.../       # D3.js visualization
│
├── Interview/                      # Interview preparation
│   ├── leetcode/
│   │   ├── Questions/              # Problem sets (PDFs, Excel)
│   │   └── Solutions/              # Python solutions (numbered by problem)
│   │       ├── N_Problem_Name.py   # Individual solutions
│   │       ├── helpers.py          # Shared utilities (TreeNode, etc.)
│   │       └── N Topic/            # Topic-grouped solutions
│   ├── LLD/                        # Low-Level Design implementations
│   │   ├── A_System_Name/          # Each system in its own folder
│   │   │   ├── demo.py             # Usage demonstration
│   │   │   ├── readme.md           # Design documentation
│   │   │   └── *.py                # Component classes
│   │   └── design patterns.md      # Pattern reference
│   ├── Behavioral/                 # Behavioral interview prep
│   ├── HLD/                        # High-Level Design (placeholder)
│   └── Notes/                      # Algorithm guides and cheatsheets
│
├── Languages/                      # Language learning (French, German, Japanese)
│   └── Language/unit N-M/          # Unit-based organization
│
├── Tutorials/                      # Video tutorial notes
│   ├── AI/                         # Andrej Karpathy, RAG tutorials
│   └── GPU/                        # NVIDIA GPU workshops
│
└── .kiro/steering/                 # AI assistant guidance
```

## Naming Conventions
- LeetCode: `{number}_{Problem_Name}.py` (underscores, title case)
- LLD systems: `A_{System_Name}/` prefix for alphabetical sorting
- Chapters: `Chapter N - Topic Name/`
- Notebooks: descriptive names with `.ipynb` extension
