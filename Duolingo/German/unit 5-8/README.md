# UNITS 5 to 8

## Unit 5 : Present Tense

### U5 : Tips

- Verbs have different endings depending on the subject of the sentence. Lucky for you, `er` and `sie` use the same ending!

| subject | verb (schwimmen/swimming) |
| :------: | :------: |
| ich | schwimm**e** |
| du | schwimm**st** |
| er / sie | schwimm**t** |
